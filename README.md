---

## **Features**

- Categories: Hentai, Japanese, Russian, Indian, Korean, African  
- Daily limit: 6 videos per 12 hours per user  
- Referral system: 3 bonus videos per referred user  
- Force-join channels before accessing videos  
- Auto-index new videos from source channels  
- Admin panel: `/stats`, `/broadcast`, `/resetlimit`  
- Logs user activity and events in MongoDB and optional LOG_CHANNEL  
- Fully deployable on Heroku  

---

## **Requirements**

- Python 3.11+  
- MongoDB database (Atlas or local)  
- Pyrogram & tgcrypto  
- Telegram Bot Token
